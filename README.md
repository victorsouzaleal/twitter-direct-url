## Instalation :
```bash
> npm i --save twitter-url-direct
```

## Example
```js
const twitterGetUrl = require("twitter-url-direct")
let response = await twitterGetUrl("https://twitter.com/papajuiceman/status/1532425868521394178")
console.log(response)
```
